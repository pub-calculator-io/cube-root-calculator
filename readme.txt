== Description ==

Cube root calculator finds the principal (real) cube root of positive and negative numbers and the imaginary cube roots of the given number.

[https://www.calculator.io/cube-root-calculator/](https://www.calculator.io/cube-root-calculator/)

__Cube Root Calculator Shortcode__

`[ci_cube_root_calculator]`

== Installation ==

1. Upload the /ci_cube_root_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the [Cube Root Calculator](https://www.calculator.io/cube-root-calculator/ "Cube Root Calculator Homepage") plugin through the "Plugins" menu in WordPress.

== Cube Root Calculator Usage ==

* Add the shortcode `[ci_cube_root_calculator]` to your page, post or sidebar.
* Add the following code: `<?php display_ci_cube_root_calculator(); ?>` to your template where you would like the Cube Root Calculator to appear.
