Cube root calculator finds the principal (real) cube root of positive and negative numbers and the imaginary cube roots of the given number.

[https://www.calculator.io/cube-root-calculator/](https://www.calculator.io/cube-root-calculator/)

== Usage ==

Add the Cube Root Calculator shortcode to your page, post or sidebar:

`[ci_cube_root_calculator]`

Add the following code to your template where you would like the Cube Root Calculator to appear:

`<?php display_ci_cube_root_calculator(); ?>`

== Installation ==

1. Upload the Cube Root Calculator /ci_cube_root_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Cube Root Calculator plugin through the "Plugins" menu in WordPress.
