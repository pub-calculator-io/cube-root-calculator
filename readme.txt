== Description ==

Cube root calculator finds the principal (real) cube root of positive and negative numbers and the imaginary cube roots of the given number.

Libraries in use:
1. https://mathjs.org/
2. https://katex.org/
3. https://github.com/aFarkas/lazysizes
4. https://github.com/RobinHerbots/Inputmask
5. https://air-datepicker.com/
6. https://www.chartjs.org/

== Installation ==

1. Upload the /ci_cube_root_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the [Cube Root Calculator](https://www.calculator.io/cube-root-calculator/ "Cube Root Calculator Homepage") plugin through the "Plugins" menu in WordPress.
3. Configure your settings via Appearance > Widgets > Cube Root Calculator.
4. And you're good to go!
